# Build a Production-Ready Food Delivery Customer Mobile Application

Create a complete food delivery customer mobile application similar to Zomato and Swiggy using React Native and Expo.

## Objective

Develop a modern, scalable, production-ready food ordering mobile application that can be published to both Google Play Store and Apple App Store.

The application should focus only on the Customer App and consume APIs from an existing Node.js/Express/MongoDB backend.

---

# Technology Stack

Frontend:

* React Native
* Expo
* React Navigation
* Redux Toolkit
* Axios
* NativeWind (Tailwind CSS for React Native)
* React Hook Form
* Async Storage

Integrations:

* Google Maps API
* Firebase Cloud Messaging (Push Notifications)
* Stripe/Razorpay Payment Integration
* Socket.io Client for Real-Time Tracking
